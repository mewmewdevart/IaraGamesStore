## 📚 Sobre o Projeto

Este repositório contém o desenvolvimento inicial da plataforma **Iara Games**, criado como parte da **Sprint 01** do projeto acadêmico.
O objetivo desta etapa é estabelecer as **bases conceituais, visuais e estruturais da plataforma**, incluindo decisões iniciais de **design, UX e organização do projeto**.
